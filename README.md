<h1 align='center'>Sobre o projeto</h1>


<p align='center'>O Solar System foi um projeto desenvolvido durante o curso de formação Trybe que tem como objetivo testar conhecimentos React.</p>

<p align='center'>Neste projeto foi desenvolvida uma página que mostra todos os planetas do sistema solar e basicamente lida com ESTADOS LOCAIS</p>
<br>
<h1 align='center'>Tecnologias utilizadas</h1>


                                                - React
                                                - Componente de Classe
                                             

<h1 align='center'>Como executar o projeto</h1>

<p align='center'>Com tudo instalado, siga os passos abaixo para executar o projeto:</p>

```bash
# Clone o repositório
$ git clone https://github.com/seu-usuario/nome-do-repositorio.git

# Acesse o diretório do projeto
$ cd nome-do-repositorio

# Faça a instalação do pacote NPM
$ npm install

# Crie sua Branch
$ git checkout -b nome-da-sua-branch

# Mão na massa!

```
